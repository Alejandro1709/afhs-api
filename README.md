## Run Locally

Clone the project

```bash
  git clone https://github.com/Alejandro1709/afhs-api.git
```

Go to the project directory

```bash
  cd afhs-api
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```


## Tech Stack

**Client:** ReactJS, Typescript

**Server:** Node, Express, TypeScript
